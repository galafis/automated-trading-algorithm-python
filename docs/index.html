Automated Trading Algorithm - Python

Este repositório apresenta um algoritmo de trading automatizado desenvolvido em Python, focado em estratégias de cruzamento de médias móveis.

Visão Geral

O projeto visa demonstrar a implementação de uma estratégia de trading simples, mas eficaz, utilizando médias móveis para gerar sinais de compra e venda. Inclui estrutura de código organizada, testes unitários e um exemplo prático de uso.
